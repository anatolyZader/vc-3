# Change Log

All notable changes to the "VibeSwitch" extension will be documented in this file.

## [1.0.0] - 2024-12-01

### Added
- Initial release of VibeSwitch
- Status bar integration with mode indicator
- Quick pick menu for mode selection
- Support for VIBE mode (autonomous AI)
- Support for DEV mode (collaborative AI)
- Command palette integration
- Keyboard shortcut (Cmd/Ctrl+Shift+M)
- Auto-reload window option
- Automatic mode detection
- File watcher for .cursorrules changes
- Default mode file creation wizard
- Visual indicators (icons and colors) for each mode
- Configuration settings for customization

### Features
- `$(dashboard) VIBE` indicator in status bar when in VIBE mode
- `$(book) DEV` indicator in status bar when in DEV mode
- `$(gear) Mode?` indicator when no mode is set
- One-click mode switching from status bar
- Manages both .cursorrules and .vscode/settings.json
- Workspace-aware operation

## [Unreleased]

- Mode switching history
- Notifications for mode-specific tips

- Team sync for mode preferences
